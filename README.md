# x6s-reboot
A simple bash script for rebooting a Netgear Nighthawk X6S Extender.

Tested on EX8000 model.

## Usage
```sh
Usage: x6s-reboot [options...] [host]
    -h            Print this help text
    -p <password> Set the username
    -t            Test connection
    -u <username> Set the password
```
